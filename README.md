# 🤖 Interview AI  

### Practice Interviews. Build Confidence. Perform Better.

Interview AI is a modern mock interview platform designed to simulate real-world interview experiences in a structured and efficient way.  
It enables users to practice under timed conditions, track their performance, and continuously improve through a personalized dashboard.  
The platform focuses on building confidence, enhancing communication skills, and preparing users for real interview scenarios.

---

## ✨ Features

-  AI-powered mock interview sessions  
-  Timer-based question system  
-  Performance tracking with detailed history  
-  Downloadable performance report (PDF)  
-  Secure authentication system  
-  Clean and responsive user interface  

---

## 🛠️ Tech Stack

- **Frontend:** React.js  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  
- **Authentication:** JWT / Google Auth  

---

## 📌 How It Works

- Users sign in to the platform  
- Start a mock interview session  
- Answer questions within a given time limit  
- Track past interviews and performance on the dashboard  
- Download performance reports in PDF format  

---

## 💡 Key Highlights

- Real-time interview simulation with timer-based flow  
- Secure user-specific data handling  
- Performance report generation in PDF format
